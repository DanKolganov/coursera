## Шаг 4. Достаём тестовое видео

Для первой проверки не нужен MUAVIC — хватит **любого короткого ролика,
где видно лицо говорящего**. Самый быстрый вариант — youtube-dl кусочек
TED-видео (там лицо хорошо видно, и потом из MUAVIC у нас будет такое же).

Установим `yt-dlp`:

```python
!pip install -q yt-dlp
```

Скачаем 10 секунд из произвольного TED-видео, например:

```python
import subprocess
url = "https://www.youtube.com/watch?v=8jPQjjsBbIc"   # TED-talk Hans Rosling, лицо в кадре
# Скачиваем mp4 в 360p (нам качество не критично, лишь бы лицо было видно)
subprocess.run([
    "yt-dlp",
    "-f", "mp4[height<=480]",
    "--download-sections", "*0:00-0:15",   # первые 15 секунд
    "-o", "sample_video.mp4",
    url,
], check=True)
print("Скачали:", subprocess.check_output(["ls", "-lh", "sample_video.mp4"]).decode())
```

> Если этот ролик недоступен — подойдёт любой другой с говорящим
> человеком, где видно лицо. Можно даже свой записать через webcam и
> загрузить через `files.upload()`.

## Шаг 6. Смотрим картинку

```python
from IPython.display import Image
Image("sample_viz.png")
```

**Что должно быть видно:**

Сверху — лента из 8 серых квадратиков 96×96, и на каждом узнаваемо
видны **губы** (а не нос, не глаз, не половина лица). От кадра к кадру
губы по-разному раскрыты — это и есть полезная информация для модели.

Снизу — тепловая карта 80×1500: цветные горизонтальные полосы,
особенно яркие в нижней части (низкие частоты — там основная энергия
речи). Видны «формантные дорожки» — кривые линии, идущие через
спектрограмму: это резонансы голосового тракта, по ним и распознаются
гласные.

**Если что-то не так:**

| Что видишь | Что значит | Что делать |
|---|---|---|
| Лента — это не губы, а нос/глаз | Перепутали индексы ландмарок | Маловероятно, но пиши — разберём |
| Лента почти чёрная | Видео слишком тёмное | Попробуй другое видео |
| Спектрограмма равномерно бледная | Аудио тихое, нормализация не сработала | Проверь, что в видео есть звук |
| Падает с `RuntimeError: ни одного лица` | Лицо в видео слишком мелкое или повёрнуто | Возьми другое видео — крупный план |
| Mediapipe вообще не импортируется | Python 3.11 несовместим | Перезапусти runtime, в Colab по умолч. 3.10 |
