# Курсовая: Audio-Visual Speech Recognition (AVSR) в реальном времени

Мультимодальная модель распознавания речи: использует одновременно **аудио** (микрофон) и **видео** (движения губ) для повышения устойчивости к шуму.

## Структура проекта

```
cursera_claude/
├── README.md              # Исходный план курсовой (большой)
├── PROJECT.md             # Этот файл — рабочая документация
├── requirements.txt       # Зависимости Python
│
├── src/                   # Весь исходный код
│   ├── data/             # Загрузка датасета, предобработка, аугментации
│   ├── models/           # Архитектура: аудио-/видео-энкодеры, fusion, голова
│   ├── training/         # Цикл обучения, loss'ы, метрики
│   ├── inference/        # Реальное время: захват аудио/видео, демо
│   └── utils/            # Общие утилиты (логирование, конфиг)
│
├── configs/              # YAML-конфиги для разных экспериментов
├── notebooks/            # Jupyter — для отладки, визуализации, EDA
├── scripts/              # Запускаемые скрипты: train.py, eval.py, demo.py
│
├── data/
│   ├── raw/             # Сырой датасет (НЕ в git — тяжёлый)
│   ├── processed/       # Предобработанные тензоры (кропы губ, спектрограммы)
│   └── manifests/       # JSON со списком файлов и транскрипций
│
├── checkpoints/          # Сохранённые веса моделей (НЕ в git)
└── docs/                 # Заметки по теории, выписки из статей
```

## Текущий этап

**Неделя 1, день 1**: настройка окружения и подготовка данных.

## Как запускать (заглушки, появятся по ходу работы)

```bash
# Обучение
python scripts/train.py --config configs/avsr_baseline.yaml

# Оценка
python scripts/eval.py --checkpoint checkpoints/best.pt --noise-snr 5

# Демо реального времени
python scripts/demo.py --checkpoint checkpoints/best.pt
```
